# ⚔️ 不打不相識 — 六年級自然科雙人對戰複習遊戲

> 仿格鬥遊戲風格的互動式複習活動，兩位同學上台同時作答，答對攻擊對方，答錯暈眩！

🎮 **[立即遊玩](https://你的帳號.github.io/你的repo名稱)**

---

## 遊戲介紹

| 項目 | 說明 |
|------|------|
| 📚 科目 | 六年級自然科 |
| ❓ 題數 | 57 題（隨機抽題，答完換新題）|
| 👥 模式 | 雙人對戰（同一台電腦）|
| ⏱ 時限 | 每題 30 秒 |
| 💥 規則 | 答對 → 攻擊對方 20HP；答錯 → 暈眩 3 秒 |

## 操作方式

| 玩家 | 按鍵 | 說明 |
|------|------|------|
| 玩家一（左） | `1` `2` `3` `4` | 選擇①②③④ |
| 玩家二（右） | `7` `8` `9` `0` | 選擇①②③④ |

> 也支援直接**點選**選項（適合觸控螢幕）

## 題目主題

- 🎵 聲音與樂器（音調、響度、音色）
- 🌿 生態系與生物多樣性
- 🌍 環境與永續（碳足跡、水足跡、再生能源）
- 🌱 外來入侵種
- ♻️ 循環經濟與氣候變遷
